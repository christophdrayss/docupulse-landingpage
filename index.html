F.A.Q

Kann ich Docupulse auch kostenlos testen?

Gerne können sie Docupulse kostenlose testen. Setzen Sie sich mit uns in Verbindung und wir richten Ihnen die Software für einen Zeitraum von 2 Monaten gratis ein.

Welche Sicherheitsmaßnahmen werden zum Schutz meiner Daten ergriffen?

Sie können wählen, ob die Dokumente auf Ihrem eigenen Sharepoint/Azure Account oder auf den Servern von Docupulse gespeichert und prozessiert werden. Docupulse verwendet Microsoft Azure als Server Host mit standardmäßig hohen höchste Sicherheitsstandards. Dazu gehören verschlüsselte Datenübertragung, sichere Speicherung und revisionssichere Archivierung der Dokumente.

Welche individuellen Lösungen bietet der Ultimate Plan?

Der Ultimate Plan umfasst alle Funktionen des Business Plans sowie maßgeschneiderte Lösungen zur Datentransformation für Prozesse, die speziell auf Ihre definierten Arbeitsabläufe abgestimmt sind. Unser Expertenteam arbeitet eng mit Ihnen zusammen, um sicherzustellen, dass Ihre speziellen Anforderungen effizient erfüllt werden.

Wie installiere ich Docupulse?

Sie installieren Dokupulse ganz einfach via Microsoft Add In Store oder als individualisierten Server für Ihre eigenen Server.
